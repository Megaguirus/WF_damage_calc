*this is a toy project and is not meant to be taken as a drop replacement for anything, i just thought some would find it useful*

# what the hell is that ?

I woke up one day and decided that i want to understand how the damage calcution and mod system in Warframe works. 
I spent an unholy amount of time playing that game, so simulating its damage calculations has always called to me-

# current features

- basic mods metadata
- basic weapons metadata for hitscan IPS weapons
- generate all combinations of the provided mods set 

# planned features

- expand the mods list (no exploits / bugs, only intended interactions)
- add faction bonuses, damage multipliers (like viral / magnetic status effects), and some ability buffs(?)
- JSON input of weapons, Excel sheets to visualize outputs
- expand the weapons definition to include projectiles, AoE and beams
- arcanes
- DoT damage 

# what i absolutely will not be adding

- kitguns, zaws, archguns and melees, i have no idea how the damage for these weapon classes is calculated and im interested in figuring them out (for now)
- unintended interactions, such as multiplicative crit multipliers
- ability buffs in general are outside the scope of the mod system that i want to implement myself without loosing my sanity
- companion weapons and mods
- multishot.

# can you contribute ?

sure lol, the more weapons filled in with their damage values for every mod combination to the JSON the better (i dont have all weapons unlocked lol)
for now, im limitting it to IPS-based hitscan weapons, your basic assault and sniper rifles, other weapon classes will be included in the future if i do not loose interest in the project all of a sudden.
